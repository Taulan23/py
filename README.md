# Универсальная программа распознавания чисел с экрана

## Описание
Кроссплатформенная программа для распознавания чисел с экрана в реальном времени с расширенными возможностями настройки и автоматической конфигурацией.

## ⚡ Новые возможности производительности (v2.2.0)
- **🚀 Быстрый режим**: Оптимизированный интерфейс с минимальными задержками
- **💾 Кэширование данных**: Ускорение загрузки и обработки информации  
- **🔄 Динамическое обновление**: Настраиваемая частота обновления экрана
- **⚙️ Гибкие настройки**: Переключение между режимами производительности

## Функции
- 🌐 **Универсальность**: Работает на Windows, macOS и Linux
- 🔍 **Автоматическое обнаружение**: Находит Tesseract OCR в системе
- 📱 Распознавание чисел с выбранной области экрана
- 💾 Автоматическое сохранение в JSON при каждом новом числе
- 🌈 **Цветные таблицы**: Красивый вывод данных с цветовым кодированием
- 📊 **Расширенная статистика**: Анализ данных с медианой, стандартным отклонением, топ-10
- 📈 **Интеллектуальный анализ**: Распределение по диапазонам, цветовая индикация
- 💾 **Экспорт данных**: Экспорт в CSV для анализа в Excel/LibreOffice
- 🎯 **Счетчики серий**: Трекинг подряд идущих результатов по пользовательским меткам
- ⏰ Сохранение временных меток для каждого числа
- 🔄 Загрузка существующих данных при запуске
- ⚙️ Гибкая система конфигурации
- 💾 Сохранение областей экрана для повторного использования
- 🎯 Улучшенный интерфейс выбора области с отображением координат

## 🚀 Быстрый старт

```bash

pip install -r requirements.txt

# Tesseract OCR (автоматически определяется программой)

# Ubuntu: sudo apt-get install tesseract-ocr
```

## ⚡ Режимы производительности

### 🚀 Быстрый режим (рекомендуется)
```bash

```
**Особенности:**
- Обновление интерфейса каждые 0.3 секунды
- Упрощенный вывод (показывает 3 строки + 5 последних чисел)
- Кэширование данных для ускорения
- Сохранение каждого 3-го изменения
- Оптимизированное сканирование (интервал 0.6с)

### 🐌 Обычный режим (полная функциональность)
```bash

```
**Особенности:**
- Полный интерфейс со всей статистикой
- Подробные таблицы и счетчики серий
- Сохранение каждого изменения
- Стандартный интервал сканирования (1.0с)

## ⚙️ Конфигурация

### Файл config.json
```json
{
  "scan_interval": 0.8,
  "history_size": 5,
  "table_display": {
    "table_format": "simple",
    "max_rows": 5,
    "refresh_interval": 0.3
  },
  "performance": {
    "fast_mode": true,
    "cache_data": true,
    "minimal_display": false
  }
}
```

### Настройка производительности
- **fast_mode**: `true` для быстрого режима, `false` для обычного
- **cache_data**: `true` для кэширования данных
- **refresh_interval**: Частота обновления экрана (секунды)
- **scan_interval**: Интервал между сканированиями (секунды)

## 🐛 Устранение неполадок

### Программа работает медленно
1. Переключитесь в быстрый режим: `python3 123456789000.py --fast-mode`
2. Очистите кэш: `python3 123456789000.py --clear-cache`
3. Уменьшите размер области сканирования
4. Увеличьте `scan_interval` в config.json

### Ошибки при запуске
1. Проверьте установку Tesseract: программа автоматически найдет путь
2. Переустановите зависимости: `pip install -r requirements.txt`
3. Удалите файл `region.json` для пересоздания области

### Проблемы с областью сканирования
1. Удалите `region.json` и выберите область заново
2. Убедитесь, что размер области больше 10x10 пикселей
3. Проверьте, что область находится в пределах экрана
