# Универсальная программа распознавания чисел с экрана

## Описание
Кроссплатформенная программа для распознавания чисел с экрана в реальном времени с расширенными возможностями настройки и автоматической конфигурацией.

## Функции
- 🌐 **Универсальность**: Работает на Windows, macOS и Linux
- 🔍 **Автоматическое обнаружение**: Находит Tesseract OCR в системе
- 📱 Распознавание чисел с выбранной области экрана
- 💾 Автоматическое сохранение в JSON при каждом новом числе
- 📊 Красивый вывод данных с помощью tabulate
- ⏰ Сохранение временных меток для каждого числа
- 🔄 Загрузка существующих данных при запуске
- ⚙️ Гибкая система конфигурации
- 💾 Сохранение областей экрана для повторного использования
- 🎯 Улучшенный интерфейс выбора области с отображением координат
- 🖥️ Отображение системной информации
- 🔧 Настраиваемые параметры обработки изображений

## Установка

### 1. Установка Tesseract OCR

#### macOS:
```bash
brew install tesseract
```

#### Ubuntu/Debian:
```bash
sudo apt-get install tesseract-ocr
```

#### Windows:
Скачайте и установите с https://github.com/UB-Mannheim/tesseract/wiki

#### Автоматическая установка (любая ОС):
Программа автоматически найдет Tesseract OCR в системе при первом запуске

### 2. Установка Python зависимостей
```bash
pip install -r requirements.txt
```

## Использование

### 🚀 Быстрая установка (macOS):
```bash
./install.sh
```

### 📋 Основные команды:

**Запуск программы мониторинга:**
```bash
./run.sh
```

**Просмотр сохраненных данных:**
```bash
./run.sh --view
```

**Настройка программы:**
```bash
./run.sh --setup
```

**Режим отладки:**
```bash
./run.sh --debug
```

**Использование пользовательской конфигурации:**
```bash
./run.sh --config my_config.json
```

**Создание демонстрационных данных:**
```bash
python3 demo.py
```

### 🎯 Использование программы:

1. **Первый запуск**: Программа автоматически найдет Tesseract OCR в системе
2. **Выбор области**: Интерактивный выбор с отображением координат курсора
3. **Сохранение области**: Выбранная область автоматически сохраняется
4. **Мониторинг**: Программа начнет распознавание и сохранение данных
5. **Выход**: Нажмите Ctrl+C для завершения

### ⚙️ Ручной запуск (если нужно):
```bash
source venv/bin/activate
python 123456789000.py [опции]
```

### 📊 Опции командной строки:
- `--view` - Просмотр сохраненных данных
- `--config FILE` - Указать файл конфигурации
- `--setup` - Настройка программы
- `--debug` - Режим отладки
- `--help` - Показать справку

## ⚙️ Конфигурация

### Параметры в config.json:
- `history_size` - количество отображаемых чисел в истории
- `scan_interval` - интервал сканирования в секундах
- `debug_mode` - режим отладки
- `output_file` - файл для сохранения данных
- `region_file` - файл для сохранения области экрана
- `tesseract_config` - настройки Tesseract OCR
- `image_processing` - параметры обработки изображений:
  - `enhance_contrast` - улучшение контраста
  - `sharpen` - заточка изображения
  - `threshold` - пороговое значение
  - `clahe_clip_limit` - лимит CLAHE
  - `clahe_tile_grid` - размер сетки CLAHE

### Пример конфигурации:
```json
{
  "history_size": 15,
  "scan_interval": 0.5,
  "debug_mode": true,
  "image_processing": {
    "threshold": 100,
    "clahe_clip_limit": 3.0
  }
}
```
